Barbarian Quest 3
==================

Foolin around while trying to build a simple roguelike.

The game is, at least for now, more about me trying out different architectures 
for the whole engine, and so will probably stay pretty lame for quite a while.

If anyone cares to try it anyway, then simply: ::

    $ git clone git@github.com:raphigaziano/barbar3.git
    $ cd barbar3
    $ bin/barbar.py
